# 海龟策略回测看板 — 产品设计文档

## 1. 产品定位

一个交互式 Web 看板，让用户通过**可视化参数调节**，实时回测海龟策略在不同股票上的表现，快速对比不同参数组合的效果。

## 2. 功能需求

### 2.1 数据选择
- 股票选择：下拉框（兆易创新、贵州茅台、中国平安、宁德时代、隆基绿能）
- 回测时间范围：日期滑条或日期选择器

### 2.2 策略参数
| 参数 | 控件类型 | 默认值 | 范围 |
|------|---------|--------|------|
| 入场通道周期 | 滑条/数字输入 | 20 | 5 ~ 120 |
| 退出通道周期 | 滑条/数字输入 | 10 | 5 ~ 60 |
| ATR 周期 | 滑条/数字输入 | 20 | 5 ~ 60 |
| 每单位风险比例 | 滑条 | 1% | 0.5% ~ 3% |
| 最大持仓单位数 | 滑条 | 4 | 1 ~ 8 |
| 加仓步长 (×N) | 滑条 | 0.5 | 0.25 ~ 1.0 |
| 初始资金 | 数字输入 | 1,000,000 | 100,000 ~ 10,000,000 |

### 2.3 绩效指标展示
- 总收益率 (%) — 卡片
- 年化收益率 (%) — 卡片
- 最大回撤 (%) — 卡片
- 夏普比率 — 卡片
- 胜率 (%) — 卡片
- 交易总笔数 — 卡片
- 止损次数 — 卡片
- 盈利因子 — 卡片 (可选)
- 最终权益 (元) — 卡片 (可选)

### 2.4 可视化图表
1. **权益曲线**: 策略权益 vs 买入持有基准 vs 最大回撤标注
2. **价格通道与交易信号**: K线 + 唐奇安通道 + 买卖标记
3. **N值(ATR)走势**: TR柱状图 + ATR折线 + 百分位分析
4. **持仓单位变化**: 时间轴上的仓位堆叠图

### 布局说明
- **左栏 320px**: 参数面板（固定宽度，可滚动）
- **右栏 1fr**: 主内容区（顶行指标卡片，下方图表 Tabs 或网格）
- 图表默认 4 张，可用 Tab 切换或 2×2 网格展示

## 4. 交互逻辑

```
用户调节参数 → 点击"运行回测" → 后端计算 → 前端更新
    ↓
实时对比: 用户可修改单个参数反复运行，观察绩效变化
    ↓
同一页面展示多组结果的对比 (如 S1 vs S2)
```

## 5. 技术栈建议

| 层 | 技术 |
|----|------|
| 前端框架 | Vue 3 + Element Plus / 或纯 Flask + Jinja2 |
| 图表库 | ECharts (支持中国股市红涨绿跌) |
| 后端 | Flask / FastAPI |
| 计算引擎 | 复用 Task-4 turtle_strategy.py + turtle_backtest.py |
| 部署 | 单机运行，Flask 内嵌 |

## 6. 数据流

```
CSV 文件 ─→ Pandas DataFrame ─→ 策略计算 ─→ 回测引擎
                                      ↓
                               绩效指标 + 图表数据
                                      ↓
                               JSON API ─→ 前端渲染
```
